# MiMo AI Demo (For MiMo Orbit Incentive)
This is a lightweight AI project based on MiMo (Xiaomi Large Model), for MiMo Orbit incentive program application.

## What this project does
- Connect MiMo API to implement chat & code generation
- Simple AI agent logic for task automation
- Batch data processing with LLM
- Local tool integration (Python + MiMo API)

## Tech Stack
- Python 3.10+
- MiMo API (mimo-v2-pro)
- requests / python-dotenv
- Git & GitHub (version control)

## How to use
1. Install dependencies: `pip install -r requirements.txt`
2. Set your MiMo API key: `export MIMO_API_KEY=your_key`
3. Run demo: `python main.py`

## Purpose
This repo proves real AI development experience for MiMo incentive program.
